
# 作业2


## 实现功能
使用 OpenCV 与 NumPy，对至少 3 幅不同类型的图像进行增强处理。图像应包含低对比度或噪声等问题。要求实现并比较以下方法：
全局直方图均衡，CLAHE，均值滤波，高斯滤波，中值滤波，一种锐化方法，“滤波→均衡” 与 “均衡→滤波” 的组合处理
要求：
至少有一个模块自行实现，不可只调用现成接口，比较不同参数对结果的影响，展示处理结果及对应直方图，对实验现象进行分析，
给出至少两项定量评价指标
提交内容： 源代码，实验报告，结果图片
## 环境配置
python

## 使用方法
1. 准备 3 张图片并命名：
   - img1.jpg：低对比度图像
   - img2.jpg：含噪声图像
   - img3.jpg：细节纹理丰富图像
2. 将图片与代码放在同一目录
3. 运行代码，自动生成结果图

## 文件结构
- image_enhancement.py  主程序
- img1.jpg、img2.jpg、img3.jpg  输入图像
- 输出：各图片对比图与直方图
