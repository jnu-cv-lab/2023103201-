## 实验一
### 实验任务
1. 使用 OpenCV 读取测试图片（`test.jpg`）
2. 输出图像基本信息（尺寸、通道数、数据类型）
3. 可视化显示原图与灰度图
4. 保存灰度图及裁剪区域结果
5. 基于 NumPy 完成像素值读取、图像裁剪操作

### 文件清单
| 类型       | 文件名               | 说明                     |
|------------|----------------------|--------------------------|
| 源代码     | `main.py`            | 实验完整代码             |
| 测试图片   | `test.jpg`           | 原始输入图片             |
| 处理结果   | `gray_test.jpg`      | 灰度图输出结果           |
| 处理结果   | `crop_test.jpg`      | 左上角100x100裁剪区域    |
| 对比图     | `result.png`         | 原图与灰度图可视化对比   |

### 核心结果
- 图像尺寸：`宽度×高度` 像素（例如：1920×1080）
- 通道数：3（RGB/BGR）
- 数据类型：uint8
- 关键操作：像素值读取、图像裁剪、色彩空间转换
